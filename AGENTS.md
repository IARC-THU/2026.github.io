## Repo essentials
- Static GitHub Pages site; no build, test, or backend tooling in repo.
- Deployment: GitHub Pages from `main` branch, `/root` folder (no build step).
- Local preview: open `index.html` directly in a browser.
- Content edits live in `index.html` (names, roles, bios, links, stats).
- Media assets live in `assets/images/`.
- Branding tweaks live in `styles.css` (colors, typography).
