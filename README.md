# THU Aerial Robotics Team Website

Static GitHub Pages site for the International Aerial Robotics Competition (IARC) Mission 10
university team.

Click [this](https://iarc-thu.github.io/2026.github.io) to view the webpage.

## Deploy to GitHub Pages
1. Create a GitHub organization (if you do not already have one).
2. Create a repository named `<organization>.github.io` under that organization.
3. Upload the contents of this folder into the repository (including `index.html`).
4. Go to **Settings → Pages** and set the source to `main` branch, `/root`.
5. Wait for deployment to finish. Your site will be available at
   `https://<organization>.github.io`.

## Update team information
- Edit `index.html` to update names, roles, bios, links, and statistics.
- Replace placeholder images in `assets/images/` with real photos, logos, and diagrams.
- Update colors and typography in `styles.css` to match team branding.

## Local preview
Open `index.html` directly in a browser. No build or backend is required.
